# GastosClaros
Sistema de gestión de gastos compartidos — Proyecto de Arquitectura de Software (UDLA).

## Módulos
- backend-core: microservicio principal (grupos, gastos, saldos)
- modulo-categoria: microservicio de clasificación de gastos (consumidor de cola)
- modulo-pago-lambda: función serverless que simula pagos/transferencias
